# Jal Tarang

Status: L1 concept packet

Authority: this repository is a V5 concept study for private review. It is not
fabrication authority, not a measured traditional tuning packet, and not a
performance-ready setup guide. Bowl count, bowl material, bowl geometry, water
volumes, mallet design, target scale, pitch values, and stage layout are all
pending measurement or reference review.

The jal tarang is a tuned idiophone made from porcelain, ceramic, glass, or
metal bowls partly filled with water and struck with light mallets. Changing the
water volume changes each bowl's pitch. The core design challenge is a stable,
repeatable tuning workflow: each bowl must be identifiable, fillable,
measurable, playable, and drainable without pretending that a first concept
packet knows the final pitches or water levels.
